# Interiores Gonzalez - Landing Page

Bienvenido a la landing page de **Interiores Gonzalez**, una empresa dedicada al diseño de interiores y construcción, fundada por Jeanniel Gonzalez y Sara de Gonzalez.

## Descripción

Este proyecto es una landing page moderna y responsiva diseñada para mostrar los servicios, proyectos y valores de "Interiores Gonzalez". La página busca conectar con clientes potenciales ofreciendo una experiencia visual atractiva y clara sobre lo que la empresa ofrece: diseño estratégico, funcionalidad y estética.

## Características

-   **Diseño Responsivo**: Adaptable a diferentes dispositivos (móviles, tablets, escritorio).
-   **Navegación Fluida**: Menú de navegación que cambia de estilo al hacer scroll y permite desplazamiento suave a las secciones.
-   **Galería de Proyectos**: Sección visual para mostrar trabajos anteriores.
-   **Formulario de Contacto**: Formulario detallado para captar información específica de los clientes (tipo de diseño, presupuesto, tipo de negocio, etc.).
-   **Sección "Sobre Nosotros"**: Información sobre los fundadores y la filosofía de la empresa.
-   **Carrusel de Imágenes**: Animación interactiva para mostrar imágenes destacadas.
-   **Animaciones**: Uso de `IntersectionObserver` para resaltar la sección activa en el menú y animaciones CSS.

## Tecnologías Utilizadas

-   **HTML5**: Estructura semántica del contenido.
-   **CSS3**: Estilos personalizados, variables CSS, Flexbox, Grid y animaciones.
-   **JavaScript (Vanilla)**: Lógica para el menú pegajoso (sticky nav), scroll spy, validación visual de formularios y carrusel de imágenes.

## Cómo Usar

1.  Clona este repositorio o descarga los archivos.
2.  Abre el archivo `index.html` en tu navegador web preferido.
3.  Navega por las diferentes secciones para explorar el contenido.

## Estructura del Proyecto

-   `index.html`: Archivo principal con la estructura de la página.
-   `style.css`: Hoja de estilos con todo el diseño visual.
-   `app.js`: Archivo JavaScript con la lógica de interacción.
-   `assets/`: Carpeta que contiene imágenes y recursos gráficos (svg, jpg).
